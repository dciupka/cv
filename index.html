Projects

Kubernetes Cluster (Rancher)

  • Problem: No reliable, self-hosted platform for deploying containerized applications at home/lab scale.
  • Solution: Built a multi-node Kubernetes cluster managed via Rancher. Deployed production-grade workloads with persistent storage, ingress, and monitoring.
  • Stack: Kubernetes · Rancher · Docker · Linux
  • Result: Stable cluster running multiple services. Full control over infrastructure with zero cloud dependency.

Smart Shutters System

  • Problem: Manual window shutters — no scheduling, no remote control, no smart home integration.
  • Solution: Fullstack app with Django REST backend and JavaScript frontend. Physical shutters controlled via MQTT messages to embedded devices.
  • Stack: Django · MQTT · JavaScript · Raspberry Pi · Docker
  • Result: End-to-end home automation: web UI + physical device control + scheduling. Fully self-hosted.

AI Camera (Raspberry Pi)

  • Problem: Need for real-time AI image processing at the edge, without relying on cloud APIs.
  • Solution: USB camera emulator that intercepts the video stream, applies AI transformations on-device, and outputs a processed feed — transparent to applications.
  • Stack: Python · Raspberry Pi · OpenCV · Linux
  • Result: Edge AI processing with no cloud dependency. Low-latency real-time video transformation.

Hardware Random Generator (Teensy)

  • Problem: Software PRNGs are not truly random — insufficient for cryptographic or security-sensitive use cases.
  • Solution: Microcontroller-based entropy source using hardware noise. Data exposed via USB interface (migrating to Ethernet) and consumed by backend services.
  • Stack: Teensy · C++ · Python · USB/Ethernet
  • Result: True hardware entropy source integrated with software backend.
